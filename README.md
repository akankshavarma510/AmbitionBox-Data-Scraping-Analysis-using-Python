# AmbitionBox-Data-Scraping-Analysis-using-Python
This project involves scraping company data from AmbitionBox and performing data cleaning and analysis using Python.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Project Workflow
1. Data Scraping:
   - Extracted company details such as ratings, reviews, and other information from AmbitionBox website.

2. Data Cleaning:
   - Removed duplicates
   - Handled missing values
   - Standardized data for analysis

3. Data Analysis & Visualization:
   - Analyzed company ratings and trends
   - Created visualizations to derive insights

## Files
- DataScraping.ipynb – Web scraping logic
- Datacleaning.ipynb – Data preprocessing
- DataVisualization.ipynb – Insights and charts
- CSV files – Dataset used

## Key Outcomes
- Identified trends in company ratings
- Generated clean datasets for further analysis
- Built visual insights from raw web data
